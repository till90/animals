50 demonstrierende Tiere — transparent (SoniTrace)
==================================================

Jedes Tier demonstriert mit Protestschild fuer etwas anderes (ironisch/absurd).
Gedacht als lustige Marker fuer QGIS o.ae.

png/  – 50 transparente PNGs, 512x512, Alpha-Kanal (freigestellt)
svg/  – 50 vektorisierte transparente SVGs (aus den PNGs getract)
index.html – Galerie-Vorschau (PNG/SVG umschaltbar, Schachbrett = Transparenz)
preview.png – Kontaktbogen aller 50

Dateinamen entsprechen den Spalten icon_jpg/icon_svg in tiere_final.xlsx
(z.B. lion_0 / penguin_22 / bat_49) — passend zum ersten Set.

QGIS: PNG oder SVG direkt als Marker-Symbol (SVG skaliert verlustfrei).

Bilder via Google Gemini (gemini-2.5-flash-image) erzeugt — KI-generiert.
Hintergrund per rembg (u2netp) entfernt, SVG per vtracer getract.
